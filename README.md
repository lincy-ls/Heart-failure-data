# Heart-failure-data

##Project Overview
This project clinical data to identify factors associated with heart failure and predict patient outcomes using statistical and machine learning techniques.

##Dataset
Source: UCI Machine Learning Repository
Heart Failure Clinical Records Dataset

##Tools Used
-R
-RStudio

##Methods
-Exploratory Data Analysis
-Hypothesis Testing
-Logistic Regression
-Odds Ratio
-ROC curve
-Model Validation
-Confusion Matrix
-Barplots

##Results
The Logistic regression model was evaluated using accuracy, sensitivity, specificity, and AUC to assess its predictive performance.
